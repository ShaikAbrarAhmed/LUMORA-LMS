# Project Documentation Portal

Welcome to the documentation portal for **LUMORA LMS**. All technical documentation, deployment guides, user manuals, and API specs reside in this directory.

## Documentation Structure

* `/setup-guide.md` - Detailed installation and local environment setup.
* `/architecture.md` - Architectural patterns, database designs, and backend hooks descriptions.
* `/frontend/` - Frontend components, design systems, and state management docs.
* `/api/` - REST API specification and custom Frappe methods documentation.
* `/deployment/` - Production deployment guides for Docker/K8s/Nginx.

## Contributing to Docs

Please follow these guidelines:
1. Write docs in clear, concise Markdown.
2. Embed diagrams using Mermaid where applicable.
3. Keep specifications up to date when modifying codebase features.
